# 초보자를 위한 파이썬 웹 스크래핑 가이드

이 프로젝트는 파이썬을 처음 접하는 분들도 쉽게 웹 스크래핑(Web Scraping)을 경험해볼 수 있도록 구성되었습니다.
웹 스크래핑이란 인터넷 상의 수많은 정보를 우리가 원하는 형태로 수집하고 가공하는 기술입니다.

## 📚 설치된 라이브러리 (Library)

웹 스크래핑을 위해 설치된 핵심 라이브러리 4가지입니다.

### 1. Requests (요청하기)
- **역할**: 인터넷 브라우저 대신 웹 서버에 "접속"해주는 **"배달원"**입니다.
- **주요 기능**: 웹페이지의 내용(HTML)을 요청하고 받아옵니다.

### 2. BeautifulSoup4 (요리하기)
- **역할**: 가져온 웹페이지 내용(HTML)을 분석하는 **"요리사"**입니다.
- **주요 기능**: HTML 태그를 해석하고 데이터를 추출하기 쉽게 만듭니다.

### 3. lxml (요리 도구)
- **역할**: BeautifulSoup이 HTML을 분석할 때 사용하는 **"고성능 칼"**입니다.
- **특징**: 처리 속도가 빠릅니다.

### 4. Pandas (정리하기)
- **역할**: 수집한 데이터를 깔끔하게 표로 정리해주는 **"정리 전문가"**입니다.
