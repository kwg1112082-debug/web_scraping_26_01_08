# 🌐 파이썬 HTTP 요청 가이드 (Requests 라이브러리)

이 문서는 웹 스크래핑의 첫 번째 단계인 **'HTTP 요청'**에 대해 초보자도 이해하기 쉽게 설명합니다. `src/http_요청.py` 파일의 내용을 기반으로 작성되었습니다.

---

## 🏗️ HTTP 요청이란?

우리가 웹 브라우저(크롬, 엣지 등) 주소창에 `www.google.com`을 입력하고 엔터를 치는 순간, 우리 컴퓨터는 구글 서버에 **"페이지 정보를 보내주세요!"**라고 요청을 보냅니다. 파이썬에서는 이 역할을 **`requests`** 라이브러리가 담당합니다.

## 🛠️ 핵심 코드 설명

### 1. 라이브러리 불러오기
```python
import requests
```
파이썬에서 웹 소통을 담당하는 전문 도구를 가져옵니다.

### 2. 접속 시도 (GET 요청)
```python
response = requests.get("https://www.google.com")
```
- `get()` 함수는 데이터를 가져오고 싶을 때 사용하는 가장 기본적인 명령어입니다.
- 실행 결과는 `response`라는 변수에 담기며, 여기에는 서버가 보내준 모든 정보가 들어있습니다.

### 3. 성공 여부 확인 (상태 코드)
서버가 응답을 주면 항상 **'상태 코드(Status Code)'**를 같이 보냅니다.
- **200**: 성공! 데이터를 잘 가져왔습니다.
- **404**: 실패! 주소가 잘못되었거나 페이지가 없습니다.
- **500**: 서버가 아파요! (서버 내부 오류)

### 4. 데이터 확인 (Text)
```python
print(response.text)
```
서버로부터 받아온 웹사이트의 원본 코드(HTML)를 텍스트 형식으로 확인합니다. 이 원본 코드를 나중에 `BeautifulSoup`으로 분석하여 원하는 정보만 골라내게 됩니다.

---

## 💡 요약
1.  **Requests**: 웹사이트에 접속해주는 배달원
2.  **Status Code (200)**: 배달 성공 여부 확인
3.  **Response Text**: 배달된 물건 (웹사이트 HTML 코드)

다음 단계에서는 이렇게 가져온 복잡한 HTML 문서에서 요리사 역할을 하는 **BeautifulSoup**을 사용해 원하는 정보만 쏙쏙 뽑아내는 방법을 배우게 됩니다!
