# 📰 AI 기반 네이버 뉴스 추출 프로젝트

이 프로젝트는 파이썬 초보자도 쉽게 웹 스크래핑(Web Scraping)의 전 과정을 이해하고 실습할 수 있도록 구성되었습니다. 네이버 뉴스 수집부터 AI 필터링, 그리고 결과물을 멋진 엑셀 파일로 저장하는 법을 배울 수 있습니다.

---

## 📚 프로젝트 폴더 구조

프로젝트가 성격별로 깔끔하게 정리되어 있습니다:

- **📁 src/**: 실제 파이썬 코드 파일들이 들어있습니다.
  - `scrap_test-ver.FINAL.py`: 필터링과 엑셀 저장이 포함된 최종 완성본 코드입니다.
  - `http_요청.py`: 웹 접속의 기본 원리를 배우는 입문용 코드입니다.
- **📁 READ/**: 각 코드에 대한 상세 한글 가이드 문서입니다.
  - `FINAL_GUIDE.md`: 최종 뉴스 스크래퍼의 작동 원리 해설서입니다.
  - `HTTP_REQUEST_GUIDE.md`: 웹 접속(HTTP) 기초 지식 가이드입니다.
  - `pip list.md`: 프로젝트에 사용된 라이브러리 목록입니다.
- **📁 excel/**: 프로그램 실행 결과로 생성된 엑셀 데이터 파일이 저장됩니다.

---

## 🚀 시작하기 (How to Start)

1.  **가상환경 활성화**:
    터미널에서 아래 명령어를 입력해 가상환경을 켭니다.
    ```bash
    .\Scripts\activate
    ```

2.  **기초 배우기**:
    `src/http_요청.py`를 실행해보고, `READ/HTTP_REQUEST_GUIDE.md`를 읽어보세요.

3.  **최종 프로그램 실행**:
    메인 프로그램을 실행하여 AI 뉴스를 엑셀로 저장해봅니다.
    ```bash
    python src/scrap_test-ver.FINAL.py
    ```

---

## 🛠 사용된 기술 (Tech Stack)

- **Requests**: 웹 서버에 정보를 요청하는 배달원
- **BeautifulSoup4**: HTML을 분석하는 요리사
- **Pandas**: 데이터를 표로 정리하는 전문가
- **Openpyxl**: 엑셀 디자인을 책임지는 디자이너
