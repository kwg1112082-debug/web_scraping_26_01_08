# 📑 네이버 뉴스 AI 필터링 및 엑셀 저장 프로그램 가이드

이 문서는 `scrap_test-ver.FINAL.py` 파일의 작동 원리와 주요 기능을 상세히 설명합니다. 이 프로그램은 네이버 뉴스를 수집하고, 특정 키워드(AI)로 필터링한 후, 가독성이 뛰어난 엑셀 파일로 저장하는 것을 목적으로 합니다.

---

## 🚀 주요 기능 (Key Features)

1.  **자동화된 뉴스 수집**: 네이버 세계 뉴스 섹션에서 실시간으로 데이터를 가져옵니다.
2.  **스마트 필터링**: 제목에 **'AI'** 또는 **'ai'**가 포함된 뉴스만 골라내어 저장합니다.
3.  **날짜 기반 파일명**: 현재 날짜를 인식하여 `naver_news_YYYYMMDD.xlsx` 형식으로 자동 저장합니다.
4.  **프리미엄 엑셀 디자인**:
    *   **제목 행**: 진한 파란색 배경과 흰색 굵은 글씨 적용
    *   **테두리**: 모든 셀에 격자 테두리 적용
    *   **가독성**: 뉴스 제목 굵게 표시, 행 높이 및 열 너비 자동 최적화
    *   **인덱스 제거**: 불필요한 첫 번째 숫자 열 제거

---

## 🛠 사용된 라이브러리 (Tech Stack)

| 라이브러리 | 역할 |
| :--- | :--- |
| **Requests** | 웹 서버에 접속하여 HTML 데이터를 가져옴 |
| **BeautifulSoup4** | HTML 태그를 분석하고 원하는 정보를 추출함 |
| **Pandas** | 수집된 데이터를 표(Dataframe) 형태로 정리 |
| **Openpyxl** | 엑셀 파일 생성 및 전문적인 디자인 스타일링 적용 |
| **Datetime** | 현재 날짜와 시간 정보를 가져옴 |

---

## 📖 코드 작동 단계 (Code Execution Flow)

### Step 1. 웹페이지 접속
- `requests.get()`을 사용하여 지정된 URL에 접속합니다.
- `try-except` 구문을 통해 네트워크 오류 발생 시 안전하게 프로그램을 종료합니다.

### Step 2. 데이터 추출
- `BeautifulSoup`의 CSS 선택자(`select`) 기능을 사용하여 신문사, 제목, 내용, 링크를 정확하게 뽑아냅니다.
- 수집된 정보는 딕셔너리(`{}`) 형태로 리스트에 저장됩니다.

### Step 3. AI 키워드 필터링
- 파이썬의 리스트 컴프리헨션 기능을 사용하여 제목에 'AI'가 들어있는 뉴스만 선별합니다.

### Step 4. 파일명 생성 및 저장
- `datetime.now()`를 활용해 오늘 날짜를 구합니다.
- `pd.ExcelWriter`와 `openpyxl` 엔진을 결합하여 스타일이 적용된 엑셀 파일을 생성합니다.
